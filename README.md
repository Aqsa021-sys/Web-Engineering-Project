# Web Engineering Project

## 📌 Overview

This project is a web-based application developed as part of a Web Engineering lab. It demonstrates the implementation of a multi-page website with proper navigation, user interaction, and frontend validation.

---

## 🚀 Features

* Multi-page website structure
* User-friendly interface
* Form validation using JavaScript
* Navigation between pages
* Responsive design using Bootstrap
* Custom UI styling

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* Bootstrap

---

## 📂 Project Structure

```
project-folder/
│── index.html
│── registration.html
│── dashboard.html
│── courses.html
│── feedback.html
│── README.md
```

---

## ⚙️ How to Run

1. Download or clone the repository
2. Open the project folder
3. Run the project by opening `index.html` in any web browser
